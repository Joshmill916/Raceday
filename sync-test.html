✅ What to check (two-device test)
  1. Open this page in two places with the same sync code (use a normal + incognito window, or two devices). Connect both.
  2. Converge: Add a driver on one → it appears on the other within ~1s. Set a result on the other → appears on the first.
  3. Late joiner: open a third window after data exists → it fills in with the current board.
  4. Empty-room seed: a brand-new sync code → the first device populates the room.
  5. No clobber: set role register on one (adds drivers) and scoring on the other (sets results) → both survive.
  6. Privacy: type in the private note → it must never show in the cloud panel.
  7. Roles: a ?sync=CODE&role=scoring link locks the device to scoring controls; tv is read-only.
  8. Reconnect: go offline briefly, make a change, come back → it re-syncs.
